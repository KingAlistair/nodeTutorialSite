What do you need to download?


1. A development environment

A development environment is a workspace that enables developers to build an application or make changes to it without affecting the live version of the software product. These changes can include maintenance, debugging, and patching. Two popular options you can choose from is Visual Studio Code and IntelliJ IDEA.

Download Visual Studio Code

Download IntelliJ

2. Node.js

Downloading Node.js gives you access to a powerful and versatile platform that can help you build robust and scalable server-side applications using JavaScript. With it you will be able to;

1. Run server-side applications: Node.js is commonly used to develop server-side applications, such as web servers, APIs, and real-time applications, making it easier for developers to build scalable and high-performance applications.

2. Access to a vast collection of packages: Node.js has a massive package ecosystem, known as npm (Node Package Manager), which contains over 1 million packages, libraries, and tools that developers can use to build their applications.

3. Use of JavaScript on the server:With Node.js, developers can write server-side code using JavaScript, a language that they are already familiar with, instead of learning a new language or technology.
