What is Express?


Express is a popular, open-source web application framework for Node.js. It provides a set of features and tools that simplify the development of web applications in Node.js. Express is designed to be lightweight, flexible, and modular, which makes it suitable for building small to large-scale web applications.

Express provides many features that make web application development easier, such as routing, middleware, and template engine support. Routing allows developers to define how the application responds to client requests for specific URLs, while middleware functions can be used to handle authentication, logging, and other common tasks. Express also supports a wide variety of template engines, including Pug, EJS, and Handlebars, which makes it easy to render dynamic HTML pages with minimal code.

Express is highly extensible and customizable, allowing developers to add third-party plugins and middleware to enhance its functionality. Additionally, it has a large and active community that provides many resources, such as documentation, tutorials, and example code.

Overall, Express is a powerful and flexible web application framework that simplifies the development of Node.js-based web applications, making it a popular choice among developers.


Let's create your first project!


1. Create a folder for your project in your chosen IDE. Let's call it myNodeSite

2. In your terminal change your directory to that folder: cd myNodeSite

3. Install express: npm install express

Installing express creates a package.json file to handle dependencies, meta data, and scripts

4. Create a file called app.js

5. Write the following code in app.js

import express from "express";
        const app = express();
        
        const PORT = 8080;
        app.listen(PORT, (error) => {
            if (error) {
                console.log(error);
                return;
            }
            console.log("Server is running on port: " + PORT);
        });

6. Run command: node app.js

7. Try it here to see the output!
