5. Serve HTML


After creating our initial endpoints and HTML pages, let's integrate them. To do this, we'll organize our index.html file in a dedicated folder and add a new endpoint to our app.js file to serve it.

1. Create a folder named "public" in your project directory

We will keep all of our static assets here that we want to server to our Front end, such as html pages, images, css, and Java Script files.

2. Copy the codes into your app.js file

This will let express know where to look for these files

app.use(express.static('public'))

This function will send our index.html file when called.

app.get("/", (req, res) => {
        res.sendFile(__dirname + "/public/index.html")
    });

3. Open http://localhost:8080/
