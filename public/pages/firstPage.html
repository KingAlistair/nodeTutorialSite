4. HTML


What is HTML?

HTML stands for Hypertext Markup Language. It is the standard language used to create web pages and websites. HTML is a markup language that uses a set of tags and attributes to describe the structure and content of a web page. When a web browser receives an HTML document, it reads the document and renders it into a visually appealing web page that users can interact with. In short, HTML is the foundation of the web and is essential for creating websites and web applications.

Here is a basic HTML template you can use for now. Create an index.html file with it. We will use it in our next chapter 4. Serve HTML.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>I'm the title</title>
</head>
<body>
<h1>I'm an h1 title</h1> 
<p>I'm a paragraph</p> 
</body>
</html>
